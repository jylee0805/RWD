# RWD

### 目的  
練習斷點的設置以及觀察Flex、float在每個斷點的變化是否需要更改  
### Language  
- HTML
- CSS
### 遇到的問題與解法
-  **course-item設為單欄後，圖片無法置中**  
	因為<img>是行內元素，所以設定邊距不會作用，只要將display修改成block即可

-  **為何有時候內部有元素外層卻不會推高度**  
	跟float有關，因為float是浮動元素，所以容器不會有高度

-  **如何讓list的原點在框內**  
	list-style-position: inside;
